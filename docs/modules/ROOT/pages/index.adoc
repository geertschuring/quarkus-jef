== Installation

If you want to use this extension, you need to add the `io.quarkiverse.jef:quarkus-java-embedded-framework` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.jef</groupId>
    <artifactId>quarkus-java-embedded-framework</artifactId>
</dependency>
----
